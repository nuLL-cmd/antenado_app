Technologies used:
* Design  XML
* Java 8
* Retrofit (For API consumption)
* OkHttpClient (Connection management)
* GsonConvertFactory (Serialization JSON)
* Carouselview (Viewing images)
* Android-SpinKit (Animated Loadings)
* Lifecycle 
* RxJava
* RoomDatabase(Data persistence)
